# 项目简介

## 核心功能
- 数据清洗：包括缺失值处理和异常值处理
- 数据预处理：包括标准化处理和归一化处理
- 数据统计：对球员工资数据进行统计和分析
- 数据可视化：使用pyecharts、pandas、numpy、matplotlib等工具进行数据可视化
- 研究结论：提供基于数据分析的研究结论

## 适用场景
- 球员：了解自己在市场中的定位，为合同谈判做准备
- 球队：制定薪资策略，更明智地决策
- 行业内企业和投资者：了解篮球行业的发展动态和竞争格局

## 技术特点
- 使用Python进行数据分析与可视化
- 利用pyecharts、pandas、numpy、matplotlib等工具
- 数据集来源于网页文件，包运行
